# Lotto Sieve / 乐筛

乐筛是一个以体彩大乐透为起点的规则化选号辅助工具。

第一版目标是拉取或导入历史开奖数据，基于用户配置的规则过滤候选号码，并输出可复盘的候选结果。

详细产品说明见：

- [docs/product.md](docs/product.md)

## MVP

- 支持体彩大乐透。
- 支持本地历史开奖数据。
- 支持基础统计指标。
- 支持第一批筛选规则。
- 支持生成候选号码和规则解释。

## 本地运行

```bash
npm install
npm run dev
```

当前开发服务可访问：

- http://127.0.0.1:4280/

## 数据同步

工作台里可以点击“同步官方数据”，从中国体育彩票官方接口拉取大乐透历史开奖数据。

- 开发服务通过 Vite 代理请求官方接口。
- 同步成功后会缓存到浏览器本地。
- 如果没有缓存或同步失败，会继续使用内置样例数据。
